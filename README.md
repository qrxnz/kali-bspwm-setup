# kali bspwm setup

## ⚒️ Installation

```sh
git clone --recursive https://github.com/qrxnz/kali-bspwm-setup  ~/.dotfiles &&\
cd ~/.dotfiles &&\
chmod +x setup.sh &&\
./setup.sh
```
